# Quran Audio Player Dashboard

This project is a professional dashboard built with React and Vite. It features:

- Quran audio player
- Hadith section
- Dua section
- Two additional sections for future content

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm run dev
   ```

## Project Structure
- `src/` — Main source code for the dashboard and components
- `.github/copilot-instructions.md` — Copilot custom instructions
